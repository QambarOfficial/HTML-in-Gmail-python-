Generate HTML DIVs for Gmail

How to Use This Tool:

  1. Step 1: Upload a CSV file using the file input below.
  2. Step 2: Select the type of template you'd like to generate (Invoice, Payroll, or Both).
  3. Step 3: Click the "Generate" button to create the HTML code.
  4. Step 4: Copy the generated code by clicking the "Copy Code" button.
  5. Step 5: Open Gmail, start composing an email, right-click, and select "Inspect" to open the developer tools.
  6. Step 6: In the developer tools, find the section for the email body, paste the copied HTML code, and you're done!
